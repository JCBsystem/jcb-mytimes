Add mood emoji mapping as a module-level constant:

```typescript
const MOOD_EMOJI: Record<number, string> = { 1: "😞", 2: "😐", 3: "🙂", 4: "😄", 5: "🤩" };
```

State: `const [showConfirm, setShowConfirm] = useState(false);`

The component renders two modes:

**Collapsed mode** (`!isExpanded`):
- Clickable container: `<div onClick={onToggle} className="py-3 px-1 cursor-pointer hover:bg-stone-50/50 rounded-xl transition-colors">`
- Text-only memories: `line-clamp-2` on text, compact
- Image memories: show image with `max-h-[200px]`, text with `line-clamp-2`
- MemoryMeta row: time, mood emoji (use `MOOD_EMOJI[memory.mood]` instead of raw number), people, tag chips

**Expanded mode** (`isExpanded`):
- Container: `<div className="py-4 px-3 bg-stone-50/80 rounded-xl -mx-1 transition-all">`
- Click handler: `onClick={onToggle}` on the main container (but NOT on interactive elements like trash button, audio player)
- Full text: NO `line-clamp`, show all text with `whitespace-pre-wrap`
- Full-size image: if present, `<img className="w-full rounded-xl" />` (no max-h constraint)
- All metadata displayed with labels:
  - Time: `formatTime(memory.eventDate)` with date: `dayjs(memory.eventDate).format("MMMM D, YYYY")`
  - Mood: full emoji + label like "😄 Mood: 4/5"
  - People: "With: Alice, Bob" using `memory.people.join(", ")`
  - Tags: full chip display (same `rounded-full bg-stone-100 px-2.5 py-0.5 text-xs text-stone-600` styling)
- Audio player: if `memory.audioUrl` exists, render `<audio controls src={memory.audioUrl} className="w-full mt-2" />`. Use the `audioUrl` field (note: the Cloud Function writes to `voice` field in Firestore but the Memory type uses `audioUrl` -- check which field name is actually stored and map accordingly).
- Transcript: if `memory.transcript` exists (or `transcribedText` from Firestore -- check the actual field), render `<p className="text-sm text-stone-500 italic mt-1">{memory.transcript}</p>`
- Trash icon: positioned at top-right of expanded view. `<button onClick={(e) => { e.stopPropagation(); setShowConfirm(true); }} className="absolute top-3 right-3 p-1.5 rounded-full text-stone-400 hover:text-red-500 hover:bg-red-50 transition-colors">` with `<Trash2 className="size-4" />` from lucide-react. The expanded container needs `relative` positioning.
- ConfirmDialog: `<ConfirmDialog open={showConfirm} title="Delete memory?" description="This memory will be permanently deleted." onConfirm={() => { onDelete(memory.id); setShowConfirm(false); }} onCancel={() => setShowConfirm(false)} />`

Import `useState` from react, `Trash2` from lucide-react, `dayjs` from dayjs, `ConfirmDialog` from `./ConfirmDialog`, `formatTime` from `@/lib/date-utils`.
